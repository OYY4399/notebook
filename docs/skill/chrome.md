# chrome

## 1. 一键重新发起请求

- 选中 `Network`
- 点击 `Fetch/XHR`
- 选择要重新发送的请求
- 右键选择 `Replay XHR`

## 2. 在控制台快速发起请求

- 选中 `Network`
- 点击 `Fetch/XHR`
- 选择 `Copy as fetch`
- 控制台粘贴代码
- 修改参数, 回车搞定

## 3. 复制JavaScript变量

- 使用 `copy` 函数, 将 `对象` 作为入参执行即可, copy(obj)
- ctrl + c

## 4.  控制台获取元素

- 通过 `Elements` 选择要调试的元素
- 控制台直接用 `$0` 访问

## 5. 控制台引用上一次执行的结果

- 使用`$_`引用上一次操作的结果, 不用每次都复制一遍

  ```javascript
  '123'.split('') // => ['1','2','3']
  $_.reverse() // => ['3','2','1']
  $_.join('') // => '321'
  ```
